Politique de Confidentialité

Dernière mise à jour : 11 janvier 2026

Numeline ("nous", "notre" ou "l'application") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons, stockons et protégeons vos informations personnelles lorsque vous utilisez notre application mobile de vérification de rappels de produits alimentaires.

1. Informations que nous collectons

1.1 Informations de compte

  • Adresse e-mail (pour l'authentification)
  • Mot de passe (stocké de manière sécurisée et cryptée via Firebase Authentication)

1.2 Données de scan

  • Codes-barres de produits scannés
  • Numéros de lot scannés
  • Marques de produits
  • Dates et heures de scan
  • Historique de vos scans

1.3 Informations d'abonnement

  • Type d'abonnement (gratuit, pack 500, illimité)
  • Nombre de scans effectués
  • Date d'expiration de l'abonnement

1.4 Données d'utilisation

  • Préférences linguistiques
  • Informations sur l'appareil (type, système d'exploitation)
  • Données de performance de l'application

2. Comment nous utilisons vos informations

Nous utilisons vos informations personnelles uniquement pour :

  • Fournir le service : Vérifier les rappels de produits alimentaires via l'API OpenFDA
  • Gérer votre compte : Authentification et gestion de votre profil utilisateur
  • Historique des scans : Stocker et afficher l'historique de vos produits scannés
  • Notifications : Vous alerter en cas de rappel d'un produit que vous avez scanné
  • Gérer les abonnements : Suivre votre utilisation et vos limites de scan
  • Améliorer l'application : Analyser l'utilisation pour améliorer nos services

3. Partage des données

Nous ne vendons, n'échangeons ni ne louons vos informations personnelles à des tiers. Vos données peuvent être partagées uniquement avec :

  • Firebase/Google Cloud : Pour l'authentification et le stockage sécurisé des données (Firebase Authentication, Firestore)
  • Google Cloud Vision API : Pour la reconnaissance optique de caractères (OCR) des numéros de lot
  • OpenFDA API : Pour obtenir les informations publiques sur les rappels de produits
  • OpenFoodFacts API : Pour obtenir les informations publiques sur les produits alimentaires

Ces services tiers sont conformes aux normes de sécurité et de confidentialité des données.

4. Sécurité des données

Nous prenons la sécurité de vos données très au sérieux :

  • Les mots de passe sont cryptés via Firebase Authentication
  • Les communications sont sécurisées par HTTPS
  • Les données sont stockées sur les serveurs sécurisés de Google Cloud (Firebase)
  • Accès limité aux données par des mécanismes d'authentification robustes

5. Conservation des données

Nous conservons vos données personnelles aussi longtemps que votre compte est actif. Vous pouvez demander la suppression de votre compte et de toutes vos données à tout moment en nous contactant.

6. Vos droits

Conformément au RGPD et aux lois sur la protection des données, vous avez le droit de :

  • Accéder à vos données personnelles
  • Rectifier vos données si elles sont inexactes
  • Supprimer votre compte et vos données
  • Exporter vos données (historique de scans)
  • Vous opposer au traitement de vos données
  • Retirer votre consentement à tout moment

7. Cookies et technologies similaires

L'application n'utilise pas de cookies. Cependant, Firebase peut utiliser des identificateurs d'appareil pour l'authentification et la gestion de session.

8. Utilisation par des mineurs

Notre application est adaptée à tous les âges. Cependant, pour créer un compte, vous devez avoir au moins 16 ans ou avoir l'autorisation d'un parent ou tuteur légal.

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. Nous vous informerons de tout changement important en publiant la nouvelle politique sur cette page et en mettant à jour la date de "dernière mise à jour".

10. Données de localisation

Nous ne collectons PAS de données de géolocalisation. L'application ne suit pas votre position géographique.
